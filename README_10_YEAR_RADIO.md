# Jungle Star Radio 10-Year Launch Pack

This pack upgrades the current `music.html` placeholder into a real Jungle Star Music / Radio system.

## Files
- `music.html` — 10-Year Anniversary music hub with tabs, library, artists, albums, playlists, archive, sponsors.
- `radio.html` — live station page with 24-hour rotation, featured tracks, favorites, and sponsor tiles.
- `submit-music.html` — positive music submission page.
- `assets/css/radio.css` — extra radio styling.
- `assets/js/radio-app.js` — search, tabs, shuffle, favorites, JSON rendering.
- `assets/data/music-library.json` — tracks/artists/albums/playlists data structure.
- `assets/img/archive/` — copied archive photos from the upload.

## Install
Copy these files/folders into your existing `junglestar.net` repo.

Then in GitHub Desktop:
1. Save everything.
2. Commit message: `Add Jungle Star Radio 10-year launch pack`
3. Push origin.
4. Wait a few minutes for GitHub Pages.

## Important
Do not upload large MP3/WAV files to GitHub Pages. Use GitHub for the website shell, pages, JSON, thumbnails, and embeds.

Host audio on Spotify, YouTube, SoundCloud, Bandcamp, Audiomack, Internet Archive, or a future streaming server.

## Archive photo rule
Only publish photos you own or have permission to use. For youth-facing Academy pages, avoid alcohol signage and crop anything that does not match the school mission.
