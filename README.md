# JungleStar.net Starter Site

Static site starter for JungleStar.net.

## Launch flow
1. Open this folder in Visual Studio Code.
2. Edit pages and links.
3. Open GitHub Desktop and add this folder as a repository.
4. Commit: `first launch structure`.
5. Publish to GitHub as a public repository.
6. In GitHub: Settings -> Pages -> Deploy from a branch -> main -> /(root) -> Save.

## Money link rule
Keep payments on dedicated pages only:
- donate.html
- sponsor.html
- shop.html
- coffee.html
- consulting.html

Do not place private keys, passwords, API secrets, or payment forms in this repo.
